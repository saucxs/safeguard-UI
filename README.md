# safeguard-UI
react脚手架，react.js组件库,

##安装
```sh
git clone https://github.com/saucxs/safeguard-UI.git

cd safeguard-UI

#安装本地依赖
npm install

#启动项目
npm start

```````
在浏览器中，输入http://localhost:3001
